ℹ️ How to Use the Map

  • Single Click: View total cost breakdown for a single grid cell.
  • Shift + Click: Hold Shift and click multiple cells to select or deselect them. The popup will dynamically sum total replacement costs across all selected grids.
  • Clear Selection: Click the red "Clear Selection" button inside the grid popup window to reset highlights.
  • Layer Legend: Use the upper-right Layers control to toggle maps and view the color scale legend.
*Excludes agriculture crops & airport runway.
