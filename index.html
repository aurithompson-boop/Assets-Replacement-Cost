Infrastructure Exposure Modelling | Tongatapu - Tonga

ℹ️ How to Use the Map

  • Single Click: View total cost breakdown for a single grid cell.
  • Shift + Click: Hold Shift and click multiple cells to select or deselect them. Total replacement costs will dynamically calculate across all selected grids.
  • Clear Selection: Click the red "Clear Selection" button in the popup to reset highlighted grids.
  • Layer Legend: Use the upper-right Layers control to toggle maps and view the color scale.
*Note: Each grid contains estimated replacement costs of all major assets, apart from agriculture (crops) and the airport runway, which are not included.
